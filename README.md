# Cronómetros Sumo y Minisumo (CSM)

## Descripción

Códigos para los *Cronómetros Sumo y Minisumo*, que forman parte del desarrollo de la [Asimov Cup](https://ieeeitba.org.ar/asimovcup) en las competencia de *Sumo* y *Minisumo*.

CSM permiten dar la señal de comienzo del asalto de manera sincronizada, cronometrar el tiempo de detención del asalto por no-movimiento o movimiento no-favorable a ningún robot, y obtener el tiempo total de duración del asalto.
